# More Maps C++

Header-only C++17 library that adds more maps!

### type_map
* `type_map<T, Map, MapArgs...>`
* `iterator`
* `const_iterator`
* `mapped_type`
* `value_type`

<br>

* `begin()`
* `end()`
* `const begin()`
* `const end()`

<br>

* `find<T>()`
* `contains<T>()`

<br>

* `insert<T>(mapped_type)`
* `insert<T...>(mapped_type)`
* `insert<T...>(mapped_type...)`

<br>

* `erase(iterator)`
* `erase(const_iterator)`
* `erase<T>()`
* `erase<T...>()`

<br>

* `get<T>()`
